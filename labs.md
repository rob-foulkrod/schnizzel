## Optional Labs

### Customize Your GitHub Copilot Experience

- **Customize GitHub Copilot's behavior with custom instructions, prompts, and custom agents**
  [Exercise Link](https://github.com/skills/customize-your-github-copilot-experience)

### Scale Institutional Knowledge using Copilot Spaces *(~30 minutes)*

- **Build a knowledge management system using Copilot Spaces for team collaboration**
  Add repositories and instructions to a Copilot Space, create issues, explore project management documentation, and update docs based on discovered gaps.
  [Exercise Link](https://github.com/skills/scale-institutional-knowledge-using-copilot-spaces)

### Integrate MCP with GitHub Copilot *(~1 hour)*

- **Connect a GitHub MCP server to Copilot Agent Mode for an end-to-end workflow**
  Set up a GitHub MCP server, delegate Copilot to research projects and open issues, then implement a feature from idea to pull request.
  [Exercise Link](https://github.com/skills/integrate-mcp-with-copilot)

### Spec Kit Lab *(~75 minutes)*

- **Implement a product feature using GitHub Spec Kit**
  Use spec-driven development with GitHub Spec Kit and GitHub Copilot to generate a constitution, specification, plan, and tasks, then implement an MVP feature for an existing .NET application.
  [Exercise Link](https://microsoftlearning.github.io/mslearn-github-copilot-dev/Instructions/Labs/LAB_AK_14_implement-spec-driven-development.html)
